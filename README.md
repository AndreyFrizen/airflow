#Архитектура

Система состоит из следующих компонентов, работающих в изолированной Docker-сети:

    Airflow: ETL-платформа с LDAP-авторизацией.

    PostgreSQL: Бэкенд БД с принудительным SSL/TLS.

    OpenLDAP: Служба каталогов с SSL для управления пользователями.

    Prometheus & Grafana: Стек мониторинга ресурсов и сервисов.

#Требования

    Ansible 2.10+

    Docker & Docker Compose plugin

    Доступ по SSH к целевой машине

```

.
├── deploy.yml            # Точка входа Ansible
├── group_vars/           # Переменные и Vault (пароли)
├── roles/                # Роли для каждого сервиса
├── scripts/              # Скрипты (бэкап)
└── templates/            # Конфигурационные шаблоны (.j2)

```
